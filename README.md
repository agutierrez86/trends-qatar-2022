# 🏆 Qatar 2022 - Google Trends URL Generator

Esta aplicación de **Streamlit** permite generar de forma masiva y dinámica las URLs de **Google Trends** para cada uno de los 64 partidos del Mundial de Qatar 2022. 

El objetivo es facilitar el análisis de datos de búsqueda (interés relativo, temas y consultas relacionadas) utilizando los **Entity IDs** oficiales de Google para mayor precisión.

## 🚀 Funcionalidades

* **Listado Completo:** Incluye los 64 partidos desde la fase de grupos hasta la final.
* **Filtros Inteligentes:** Filtra por fase del torneo (Octavos, Cuartos, etc.).
* **Segmentación Geográfica:** Cambia el país de origen de las búsquedas (GEO) para comparar comportamientos entre mercados (ej. AR, MX, ES).
* **Precisión Técnica:** Utiliza identificadores de entidad (`/m/...`) para evitar ruido de palabras clave genéricas.
